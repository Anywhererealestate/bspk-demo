# Contributing to @bspk/ui

Thank you for considering contributing to @bspk/ui! We appreciate your help in improving our React component library.

## How to Contribute

**Request Access**: Since our repositories are currently private, please contact us at <a target="_blank" href="mailto:bspk.dev@anywhere.re">bspk.dev@anywhere.re</a> to request access.

**Clone the repository**

```bash
git clone https://github.com/Anywhererealestate/bspk-ui.git
```

**Create a new branch**:

```bash
git checkout -b feature/your-feature-name
```

**Make your changes**: Implement your feature or bug fix.

**Stage and commit your changes**: Use [semantic versioning](https://semver.org/) for your commit messages.

```bash
git add .
git commit -m "feat: Description of your feature"
```

**Push to your branch**:

```bash
git push origin feature/your-feature-name
```

**Create a Pull Request**: Go to the @bspk/ui repository and click "New Pull Request". Provide a clear description of your changes.

## Reporting Issues

If you find a bug or have a feature request, please email us at <a href="mailto:bspk.dev@anywhere.re">bspk.dev@anywhere.re</a> with a detailed description.

Thank you for your contributions!
